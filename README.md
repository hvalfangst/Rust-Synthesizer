# Rust Synthesizer


## Requirements

* x86-64
* Linux/Unix
* [Rust](https://www.rust-lang.org/tools/install)

## Cargo dependencies
* rodio = "0.17.1"
* console = "0.15.7"
* rand = "0.8"

## Startup

The script "up" builds and runs our application by eexecuting the following:
```
1. cargo build
2. cargo run
```
